# AI / Data / AWS Portfolio

Static GitHub Pages portfolio site with:
- multi-blog layout
- AI architecture diagrams
- AWS service diagrams
- code examples
- syntax highlighting

## Files
- `index.html`
- `style.css`
- `script.js`

## Deploy on GitHub Pages
1. Push these files to your GitHub repository
2. Go to **Settings > Pages**
3. Under **Source**, choose **Deploy from branch**
4. Select **main** branch and **/root**
5. Save

Your site will be published on GitHub Pages.
